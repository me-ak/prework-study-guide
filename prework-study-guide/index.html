Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The alt attribute contains a text string that describes the sppearance and functionality of the image.
  • When using heading, it is important to use the heading tags in order from h1 to h6
  • Separate the webpage with 'header', 'main', 'section' and 'footer' tags

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class can have CSS styles by adding the class name as the selector, and adding a period before the name.
  • We can apply a CSS rule to more than one selectors by separating them each with a comma
  • Using * as a selector, indicates you are going to apply the style to all the elements visible on the page
  • To link a stylesheet to a HTML document, add [link rel='stylesheet' href='./pathtothestylesheet/style.css'> to the html [head>

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git log: all commits can be viewed
  • git add: add file to staging area
  • git commit -m "": creates a snapshot of your repo, leave a message describing the changes that were made
  • git init: turn a directory into a repository
  • git reset [SHA ]: go back to a previous commit
  • git merge: merge 2 branches

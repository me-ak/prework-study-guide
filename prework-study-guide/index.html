HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The alt attribute contains a text string that describes the sppearance and functionality of the image.
  • When using heading, it is important to use the heading tags in order from h1 to h6
  • Separate the webpage with 'header', 'main', 'section' and 'footer' tags

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class can have CSS styles by adding the class name as the selector, and adding a period before the name.
  • We can apply a CSS rule to more than one selectors by separating them each with a comma
  • Using * as a selector, indicates you are going to apply the style to all the elements visible on the page
  • To link a stylesheet to a HTML document, add [link rel='stylesheet' href='./pathtothestylesheet/style.css'> to the html [head>

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git log: all commits can be viewed
  • git add: add file to staging area
  • git commit -m "": creates a snapshot of your repo, leave a message describing the changes that were made
  • git init: turn a directory into a repository
  • git reset [SHA ]: go back to a previous commit
  • git merge: merge 2 branches

JavaScript

    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
    • Functions are the building blocks of JavaScript. When we create a function, we are defining a block of code that performs a specific task and giving it a unique name. However, just defining a function does not mean it will execute in our code automatically. To execute a function, we must call it in our code.
    • Math.floor and Math.random are both properties of Javascripts built-in Math objects that can be used together to generate a random number
    • To create a function, we use the keyword function followed by a name for the function and parentheses. function nameoffunction ();
    • To call a function, we write: nameoffunction ();
    • To find the length of an array, we write: nameofarray.length
    • The for loop is split into 3 statements: 1) the starting point for the loop 2) the condition 3) what allows the array to iterate over each time
    • Example of a for loop: for (var x=0; x< shapes.length; x++){console.log(shapes[x]);}
    • To create a variable: var nameofvariable = ["first", "second", "third", "fourth"];
    • Arrays are zero-indexed, this means that the first item in the array has an index number of 0
    • If we want to access a specific item in our array: name of variable[itemindexnumber];
    • To create an if statement: if (condition) { //block of code that will be executed if the condition is truthy}
    • We can have more than one condition by using else if
    • To connect the JavaScript file to the HTML file, write: < script src="./pathtofile/script.js"> just above the tag in the html file
    • Control flow is the order in which a computer executes code in a script, code runs from the top of the page down to the bottom of the page
